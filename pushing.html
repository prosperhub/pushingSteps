pushing command to Github

  1. git clone (then paste the link of The Github Repository)
  2. git add .(to add all files and folders you want to push to github)
  3. git status (to see the changes)
  4. git commit -m "you put the comment in this quotation"
  5. git push (for pushing to the github your codes this is the final step, ou can also put git push origin then tou put the name of the branch)

git init

git add README.md

git commit -m "first commit"

git branch -M main

git remote add origin https://github.com/prosperhub/pushingLesson.git

git push -u origin main
